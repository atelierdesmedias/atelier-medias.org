## Deploy 

We only work on `master` branch.

### Deploy `staging`

Use deployer script to push master on staging: 

```shell
$ deployer/push-master-to-staging.sh
```

### Deploy `production` 

Use deployer script to push master on production: 

```shell
$ deployer/push-master-to-staging.sh
```
